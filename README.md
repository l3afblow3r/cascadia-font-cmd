# cascadia-code-cmd
Make Cascadia Mono Usable In Original Command Prompt

# Get Cascadia Font
https://github.com/microsoft/cascadia-code

# How to apply?
Double-click `cascadia-font-cmd.reg` then click `Yes` and `OK`.
